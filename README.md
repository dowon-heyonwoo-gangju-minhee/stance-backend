# stance-backend

#### 프로젝트 구성요소

언어: Java 21 (Amazon Corretto)

프레임워크:
- Spring Boot 3.3.2

라이브러리:
- Spring Boot Starter Actuator
- Spring Boot Starter Data JPA
- Spring Boot Starter Web
- Lombok
- Spring Boot DevTools
- Swagger
- JUnit (테스트용)

데이터베이스:
- H2 Database (개발/테스트용 인메모리 데이터베이스)
- MySQL (MySQL Connector/J 사용)

빌드 툴:
- Gradle

빌드 언어:
- Kotlin DSL
